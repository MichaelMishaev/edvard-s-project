# Jerusalem Quest - משחק חידות על ירושלים

## על הפרויקט

**Jerusalem Quest** הוא משחק חינוכי אינטראקטיבי בנושא ירושלים, שנבנה עבור תלמידי כיתה ו'1 בבית ספר **בארי**.

המשחק פותח על ידי **אדוארד** כפרויקט לימודי לכיתה.

## מה המשחק עושה?

התלמידים עונים על חידות בנושא ירושלים, צוברים נקודות ותגים, ומתחרים על מקום בטבלת המובילים של הכיתה.

### תכונות עיקריות

- **חידות על ירושלים** - שאלות קצרות ומהנות על העיר
- **מערכת ניקוד** - תשובה נכונה = 10 נקודות, בונוס מהירות = +5 נקודות
- **תגים (Badges)** - פרסים מיוחדים על הישגים:
  - מומחה ירושלים - כל התשובות נכונות
  - חושב מהיר - סיום במהירות
  - כוכב היסטוריה - תשובות נכונות על שאלות היסטוריות
  - חוקר חכם - 5 תשובות נכונות
  - גיבור השתתפות - סיום המשחק
- **טבלת מובילים** - דירוג כל השחקנים עם הדגשה צבעונית ל-5 הראשונים
- **בטיחות** - סינון שמות, ללא צ'אט, שליטת מורה מלאה

### איך זה עובד?

1. התלמיד מכניס שם ולוחץ "התחל משחק"
2. עונה על חידות (3 אפשרויות לכל שאלה)
3. רואה ניקוד סופי, תשובות נכונות ותגים שהרוויח
4. צופה בטבלת המובילים של הכיתה

## טכנולוגיות

- **צד לקוח:** React
- **צד שרת:** Node.js + Express
- **מסד נתונים:** PostgreSQL
- **שפה:** עברית מלאה עם תמיכה ב-RTL

## הרצה מקומית

```bash
# התקנת תלויות
npm install

# הרצת השרת והלקוח
docker-compose up
```
